# cn-zh-guide

本仓库用于归档和发布多个独立 HTML 页面。

## 项目简介

这是一个以中文内容为主的静态页面归档仓库。仓库内收录了若干独立编写的 HTML 文件，涵盖不同主题与用途。所有页面均可在浏览器中直接打开查看，无需额外依赖或后端服务。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库首页 / 导航页
└── pages/             # 归档页面存放目录
    └── ...            # 各独立 HTML 文件
```

- `index.html`：可选入口页面，可提供内部链接索引或简单说明。
- `pages/`：存放所有归档的 HTML 文件，每个文件均为独立页面，互不依赖。

## 页面归档说明

- 每个 HTML 文件均为自包含页面，内嵌样式与脚本，无需外部资源。
- 文件命名采用英文或拼音，便于 URL 识别。
- 页面内容以中文为主，部分可能包含少量英文术语或代码示例。
- 归档内容不针对任何特定域名或网站，仅作为资料整理与备份。

## 维护说明

- 本仓库不定期更新，新增或修改页面会直接提交至主分支。
- 欢迎通过 Issue 提交建议或报告问题。
- 如需引用或分发页面内容，请保留原始文件完整性。
